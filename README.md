# Mahbub Academic Portfolio (Vite + React + Tailwind)

This is a ready-to-upload portfolio website project.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages (simple)

1) Create a GitHub repo (example: `mahbub-portfolio`)

2) Push your code:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/<YOUR_USERNAME>/<YOUR_REPO>.git
git push -u origin main
```

3) Deploy:

```bash
npm run deploy
```
